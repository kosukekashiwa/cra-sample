# cra-sample

An app created by "npx create-react-app".
An app to check the technologies I'm interested in, such as Recoil.

- React 17.0.2
- TypeScript 4.5.2
- react-router-dom 5.3.0
- axios 0.25.0
- Recoil 0.6.1
- MUI 5.1.1
- Storybook 6.4

# Usage

1. Start json-server.

```bash
$ npm run mock
```

2. Start the React app.

```bash
$ npm start
```

3. Check the Storybook.

```bash
$ npm run storybook
```
